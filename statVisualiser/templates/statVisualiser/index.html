Stats & Stuff

By Chris and Luke

Welcome to stats n stuff, a webapp which allows you to visualise some basic statistical distributions:


Distributions

Allows for rapid visualisation of some given distributions, some basic computations surrounding each model, allows for an examination of the convolution between two independent random variables.


The Law of Large Numbers

A demonstration of applying the law of large numbers to the binomial distribution. The demonstrations include the normal approximation, and the poisson approximation (the law of rare events).


Generate Dataset

Lets you generate your own dataset based on the chosen distributions. This can then be downloaded in a csv format for your own use. You can generate up to 2 variables, with an option to visualise the convolution of the two variables.

Glossary of figures available
  • PMF: Probability Mass Function. A function for discrete variables that returns the probability of a given event occurring.
  • PDF: Probability Density Function. This is analogous to the PMF for continuous variables, however it indicates the probability density, not the probability of a given outcome.
  • CDF: Cumulative Density Function. This returns the cumulative probability for a given input i.e. the probability the variable will be less than the input value.

    For the discrete case, the CDF function sums over all values of the PMF up to the input. For the continuous case, it integrates over the variable's region to the value.

  • Supported region: Shows the values a given distribution can occupy with a non-zero probability.
  • Joint Probability Functions: The multi-dimensional case of the PDF and CDF. This website supports up to 2 independent variables. The PDF is a product of each variable's PMF/PDF.
